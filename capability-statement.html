Best-fit blockers

Build and release failures

  • EAS Build failures on iOS or Android.
  • TestFlight build installs but crashes on launch.
  • App Store submission or metadata blockers.
  • CI/CD release flow breaks after dependency or SDK changes.

Runtime and native issues

  • CocoaPods, Gradle, Xcode, or native module conflicts.
  • Release-only crashes that do not reproduce in Expo Go.
  • Push notification, WebView, deep-link, or config-plugin failures.
  • Expo SDK and React Native upgrade regressions.

What a diagnostic needs

  • Expo SDK and React Native versions.
  • Target platform and exact release deadline.
  • EAS profile, build logs, crash logs, screenshots, and App Store Connect or TestFlight error text.
  • Repository access or a minimal reproduction when available.
  • List of native modules, config plugins, and recent dependency changes.
