Diagnostic checklist for stuck releases

Expo / EAS build failures

Install errors, prebuild issues, config plugins, native modules, or failed cloud builds.

TestFlight / App Store

ITMS errors, archive export, app thinning, metadata, assets, and submission blockers.

Dependency conflicts

Version mismatches, peer deps, native module errors, Metro issues, and config drift.

Push notifications

FCM, APNs, permissions, tokens, background handlers, and release-only failures.

WebView issues

Cookies, deep links, content blocking, auth callbacks, in-app browser, and crashes.

iOS / Android native setup

Capabilities, signing, entitlements, Gradle, manifests, permissions, and profiles.

Production bugs

Release-only crashes, edge cases, small hotfixes, runtime behavior, and verification.
